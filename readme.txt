●このソフトの概要
PSP用の2ちゃんねるブラウザです。
3.xxカーネルで動作し、公式OSKが使用可能です。

●動作環境・インストール
CFW 3.52M33以上で動作します。
解凍してでてきたファイルを ms0:/PSP/GAME/owata/ の中に入れて下さい。
CFWのリカバリーメニューからGAMEフォルダが3.xxカーネルで動作するように設定して下さい。
*
バージョン0.6.0からフォントファイルが別になったためfontフォルダとその中にフォントファイルが必要です。
解凍したファイルには標準のモナーフォントとnaga10フォントが同梱されています。
*
バージョン0.7.0からカラー設定ファイルをcolorフォルダに変更しました。
colorフォルダ内に複数の設定ファイルを置くとSTARTメニューで変更できます。
"color.ini"ファイルが起動時に読み込まれるカラー設定ファイルになります。
他のファイルは任意のファイル名でOKです（拡張子も何でもいいです、なくても可）。
*
バージョン0.7.2からカラー設定ファイルの仕様が変わりました。
0xFFBBGGRR(32bits) => 0xABGR(16bits)

●自分でフォントを追加する場合
フォントファイルはシングルバイト用とマルチバイト用の2種類必要です。
config.iniで下記の項目を設定します。
//////////////////////////////////////////////////////////////////////////////
// フォント設定は以下の項目をタブ区切りで
// フォント名（半角スペースは使えません）
// シングルバイト(ASCII, 半角カナ)フォントファイル名（半角スペースは使えません）
// マルチバイトフォントファイル名（半角スペースは使えません）
// フォントの高さ(15以下)
// 行の高さ(10以上、16以下)
//////////////////////////////////////////////////////////////////////////////
CFG_FONT_SET = "IPAモナーP	IPAMonaPGothic12x13A.bin	IPAMonaPGothic12x13J.bin	13	13"

●外部板の追加
2ちゃんねる互換の掲示板（subject.txtとdat形式が同じ）を追加できます。
logフォルダに2channel.brdと同じ形式のmy.brdを置くとリストに追加されます。
my.brdを使わずに2channnel.brdを直接編集しても可能ですが再取得すると上書きされ消えてしまいます。

●PCで取得したdatファイルの表示
PCの2ちゃんねる専用ブラウザで取得したdatファイルを適当な板フォルダにコピーすることで表示できます。
フォルダにコピーする際に同じ数字のidxファイルがあれば削除してください。

●使用方法

以下のボタン設定はデフォルトのものです。

【起動画面】
	・右下に現在使用中のバージョンが表示されています。
	STARTボタン： カテゴリ・板一覧に移動します。
	○ボタン： お気に入り板(スレ)に移動します。
	×ボタン： 終了の確認ダイアログが出ます。
	ms0:/PSP/GAME/owata/log/2channel.brd が存在しない場合、ネットワークに接続して取得します。

【カテゴリ・板一覧】
	○ボタン： カテゴリを選択中は板一覧にフォーカスが移ります。
		板を選択中はその板の subject.txt を取得しスレ一覧を表示します。
	△ボタン： 2channel.brdを更新します。
	×ボタン： 終了の確認ダイアログが出ます。
	□ボタン： お気に入り板(スレ)に移動します。
	R+△ボタン： お気に入り板に登録します。
	R+□ボタン： 2ちゃんねる検索を実行します。OSKを用いて検索ワードを入力し、□で決定してください。

【スレ一覧】
	○ボタン： 選択中のスレッドを表示します。
	×ボタン： カテゴリ・板一覧に戻ります。
	△ボタン： 表示中の板の subject.txt を更新します。
	□ボタン： お気に入り板(スレ)に移動します。
	R+○ボタン： スレ一覧のソートができます。(既読順 番号順 作成順)
	R+△ボタン： スレタイの検索を実行します。
	R+□ボタン： 2ちゃんねる検索を実行します。

【お気に入り板(スレ)】
	○ボタン： 選択中のスレッドを表示します。
	×ボタン： カテゴリ・板一覧を表示します。
	△ボタン： お気に入り板(スレ)に切り替えます。
	□ボタン： 選択中のスレッドをお気に入りから削除します。(タイトルのみの削除であり、ログは削除されません。)
	R+□ボタン： 2ちゃんねる検索を実行します。
	R+○ボタン(スレ)： お気に入りに登録されているスレッドを並び替えます（板名 スレタイ 作成日）。
	R+△ボタン(スレ)： お気に入りに登録されているスレッドを一括更新します。
	更新後はLANが切断されるようになっています。

【スレッド表示画面】
	[カーソルモード]
	カーソル(0)：アナログスティックでポインタが動きます。Rボタンを押している間はポインタの動きがゆっくりになります。
	↑↓キーでスクロール←→キーでページスクロールします。
	カーソル(1)：↑↓キーでポインタ移動。←→キーでポインタの対象がアンカー レス番 IDと変わります。
	アナログスティックでスクロールします。

	[画面モード]
	ワイド(0)：画面右端で文が折り返します。
	ワイド(1)：画面幅に関係なく1024ドットで折り返します。アナログスティックで横スクロールします。

	○ボタン： フォームが立ち上がり、表示中のスレッドに書き込みできます。
	(スレッド表示画面にもどっても書き込むまで内容は保存されます。)
	×ボタン： スレ一覧に戻ります。表示位置は記憶します。
	△ボタン： 表示中のスレッドを更新します。
	□ボタン： 表示中スレッドのログを削除し、スレ一覧に戻ります。
	R+○ボタン： 選択中のスレッドをお気に入りに登録します。
	R+□ボタン： 選択中のスレッドをお気に入りから削除します。
	R+△ボタン： カーソルモード（上記）を切り替えます。
	R+×ボタン： 画面モード（上記）を切り替えますす。

	レスアンカ(>>347など)にポインタが乗った状態で、
	○ボタン： ウィンドウが立ち上がりそのレス番の内容を表示します。
	△ボタン： そのレス番まで表示位置を移動します。

	レス番にポインタが乗った状態で、
	○ボタン： そのレス番にむけたレスアンカがついた状態で書き込みフォーム
	が立ち上がります。

	IDにポインタが乗った状態で、
	○ボタン： そのIDのレスを抽出します。
	□ボタン： そのIDのレスをスレッドから隠すことができます。

	URIにポインタが乗った状態で、
	○ボタン： リンク先の状態により動作が異なります。
	リンク先が対応している画像の場合、表示します。
	リンク先がそれ以外の場合はテキストとして表示します。
	（表示中○ボタンで文字エンコードを変更できます。）
	画像その他とも一時的にCACHEフォルダに保存されますが削除可能です。

【画像表示画面】
	○ボタン： 表示中の画像を拡大縮小します。
	×ボタン： CACHEフォルダから削除せずにスレッド表示画面に戻ります。
	□ボタン： CACHEフォルダから削除し、スレッド表示画面に戻ります。
	△ボタン： 下のメニューバーの表示・非表示を切り替えます。

【STARTメニュー】
	・起動画面以外でSTARTボタンを押すと小さなメニューが開き以下の設定が出来ます。

	【NG設定】
	NG名前、NGID、NGワード、NGメール
	設定の内容は、ms0:/PSP/GAME/owata/log/ 下に保存されます。

	【LAN切断】
	LANを切断します。

	【フォント設定】
	表示するフォントを切り替えます。
	メニュー外に反映されるのはメニューを閉じてからになります。

	【カラー】
	カラー設定を切り替えます。
	メニュー外に反映されるのはメニューを閉じてからになります。

【各設定】
button.ini color.ini config.ini を編集することでそれぞれ、ボタン変更、
色変更、動作変更が設定できます。詳細は各ファイルに書いてあります。
