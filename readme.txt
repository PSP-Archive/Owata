●このソフトの概要
PSP用の2ちゃんねるブラウザです。
3.xxカーネルで動作し、公式OSKが使用可能です。

●動作環境・インストール
CFW 3.52M33以上で動作します。
解凍してでてきたファイルを ms0:/PSP/GAME/owata/ の中に入れて下さい。
CFWのリカバリーメニューからGAMEフォルダが3.xxカーネルで動作するように
設定して下さい。
バージョン0.6.0からフォントファイルが別になったためfontフォルダとその中にフォントファイルが必要です。
解凍したファイルには標準のモナーフォントとnaga10フォントが同梱されています。

●自分でフォントを変更する場合
フォントファイルはシングルバイト用とマルチバイト用の2種類必要です。
config.iniで下記の項目を設定します。
CFG_FONT_FILEA = "monafontA.bin" // シングルバイト(ASCII, 半角カナ)フォント
CFG_FONT_FILEJ = "monafontJ.bin" // マルチバイトフォント
CFG_FONT_HEIGHT = 12 // フォントの高さ(15以下)
CFG_FONT_PITCH = 13 // 行の高さ(CFG_FONT_HEIGHTまたは10以上、16以下)

●使用方法

以下のボタン設定はデフォルトのものです。

STARTボタン： 小さなメニューが開き、NG設定(後述)、LAN切断ができます。

【起動画面】
・右下に現在使用中のバージョンが表示されています。
STARTボタン： カテゴリ・板一覧に移動します。
ms0:/PSP/GAME/owata/log/2channel.brd が存在しない場合、ネットワークに
接続して取得します。
○ボタン： お気に入り板(スレ)に移動します。
×ボタン： 終了の確認ダイアログが出ます。

【カテゴリ・板一覧】
カテゴリを選択し○ボタンを押すことで板一覧にフォーカスが移ります。
板を選択し○ボタンを押すとその板の subject.txt を取得しスレ一覧を表示
します。
△ボタン： 2channel.brdを更新します。
×ボタン： 終了の確認ダイアログが出ます。
□ボタン： お気に入り板(スレ)に移動します。
R+△ボタン： お気に入り板に登録します。
R+□ボタン： 2ちゃんねる検索を実行します。OSKを用いて検索ワードを入力し、
□で決定してください。

【スレ一覧】
○ボタン： 選択中のスレッドを表示します。
×ボタン： カテゴリ・板一覧に戻ります。
△ボタン： 表示中の板の subject.txt を更新します。
□ボタン： お気に入り板(スレ)に移動します。
R+○ボタン： スレ一覧のソートができます。(既読順 番号順 作成順)
R+△ボタン： スレタイの検索を実行します。
R+□ボタン： 2ちゃんねる検索を実行します。

【お気に入り板(スレ)】
○ボタン： 選択中のスレッドを表示します。
×ボタン： カテゴリ・板一覧を表示します。
△ボタン： お気に入り板(スレ)に切り替えます。
□ボタン： 選択中のスレッドをお気に入りから削除します。(タイトルのみの
削除であり、ログは削除されません。)
R+□ボタン： 2ちゃんねる検索を実行します。
R+△ボタン： お気に入りに登録されているスレッドを一括更新します。
更新後はLANが切断されるようになっています。

【スレッド表示画面】
アナログスティックでポインタが動きます。Rボタンを押している間は
ポインタの動きがゆっくりになります。ページスクロールもできます。

○ボタン： フォームが立ち上がり、表示中のスレッドに書き込みできます。
(スレッド表示画面にもどっても書き込むまで内容は保存されます。)
×ボタン： スレ一覧に戻ります。表示位置は記憶します。
△ボタン： 表示中のスレッドを更新します。
□ボタン： 表示中スレッドのログを削除し、スレ一覧に戻ります。
R+○ボタン： 選択中のスレッドをお気に入りに登録します。
R+□ボタン： 選択中のスレッドをお気に入りから削除します。

レスアンカ(>>347など)にポインタが乗った状態で、
○ボタン： ウィンドウが立ち上がりそのレス番の内容を表示します。
△ボタン： そのレス番まで表示位置を移動します。

レス番にポインタが乗った状態で、
○ボタン： そのレス番にむけたレスアンカがついた状態で書き込みフォーム
が立ち上がります。

IDにポインタが乗った状態で、
○ボタン： そのIDのレスを抽出します。
□ボタン： そのIDのレスをスレッドから隠すことができます。

URIにポインタが乗った状態で、
○ボタン： リンク先の状態により動作が異なります。
リンク先が対応している画像の場合、表示します。(一時的にCACHEフォルダ
に保存されますが削除可能です。)
リンク先がそれ以外の場合はCACHEフォルダに保存します。

【画像表示画面】
○ボタン： 表示中の画像を拡大縮小します。
×ボタン： CACHEフォルダから削除せずにスレッド表示画面に戻ります。
□ボタン： CACHEフォルダから削除し、スレッド表示画面に戻ります。
△ボタン： 下のメニューバーの表示・非表示を切り替えます。

【各設定】
button.ini color.ini config.ini を編集することでそれぞれ、ボタン変更、
色変更、動作変更が設定できます。詳細は各ファイルに書いてあります。

【NG設定】
NG名前、NGID、NGワード、NGメール
すべての画面からSTARTボタンを押すことで設定できます。
設定の内容は、ms0:/PSP/GAME/owata/log/ 下に保存されます。
