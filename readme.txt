ｵﾜﾀブラウザ
PSP用の2ちゃんねる専用ブラウザです。
CFW 3.52M33以上で動くと思います。
解凍してできたEBOOT.PBPを/PSP/GAME/フォルダに好きな名前のフォルダを作ってそこに入れてください。
操作法は画面に出てきますが画面スクロールは上下ボタンで行います。
また左右ボタン、アナログパッドでページスクロールもできます。
レス表示画面では矢印カーソルが表示されアナログパッドで操作できます。
Rボタンを押すとカーソルのゆっくり移動ができます。
カーソルを>>1のようなレスアンカーに合わせるとレス表示またはレスへ移動ができます。
urlアンカーで○ボタンを押すと対象ファイルをキャッシュフォルダに保存します。
urlがJPG、PNG画像の場合は画像が表示されます。
画像のスクロールは上下左右キー、またはアナログパッドです。
検索で英文字は大小文字を区別しないようにしました。
EBOOT.PBPと同じ位置にcolor.iniを置くと色を変更できます。
STARTボタンでメニューウィンドウが表示されます。
